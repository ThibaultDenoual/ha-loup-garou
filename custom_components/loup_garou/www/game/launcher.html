Principe

Loup-Garou est un jeu de rôle et de déduction sociale. Deux camps s'affrontent :

  • Les Loups-Garous cherchent à éliminer tous les villageois.
  • Le Village cherche à identifier et éliminer tous les loups.
Déroulement
  1. Chaque joueur reçoit un rôle secret.
  2. La Nuit — les rôles spéciaux agissent dans l'ordre (Cupidon → Voyante → Loups → Sorcière…).
  3. Le Jour — le village discute et vote pour éliminer un suspect.
  4. On recommence jusqu'à la victoire d'un camp.
Victoire
  • Loups — Les loups sont en nombre ≥ aux villageois restants.
  • Village — Tous les loups sont éliminés.
  • Amoureux — Les deux amoureux sont les seuls survivants.
